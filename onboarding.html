📧 Gmail & Google Workspace Setup Before April 20
Log into hfan@cvc.edu Gmail
Set up email signature
Settings → General → Signature
Enable keyboard shortcuts
Settings → General → Keyboard shortcuts ON. Key: e=archive, r=reply, c=compose
Enable Templates
Settings → Advanced → Templates → Enable
Create all 10 email templates
See templates below. Compose → ⋮ → Templates → Save draft as template
Set up Multiple Inboxes
Settings → Inbox type → Multiple Inboxes. Section 1: label:@action, Section 2: label:@waiting-for, Section 3: is:starred
Create Gmail labels: @action, @waiting-for, @reference, @read-later
Color: red for @action, yellow for @waiting-for, blue for @reference, gray for @read-later
Create Gmail filters for Jira notifications
From: contains "jira" or "atlassian" → Label: Jira, Skip inbox
Create Gmail filters for Basecamp notifications
From: contains "basecamp" → Label: Basecamp, Skip inbox
Set profile photo across all Google apps
